🔐 Konfigurasi Firebase

📋 Langkah-langkah Setup Firebase:

  1. Buka Firebase Console
  2. Pilih project Anda atau buat project baru
  3. Buka Build → Realtime Database dan buat database
  4. Pilih lokasi: asia-southeast1
  5. Buka Project Settings → General → Your apps → SDK setup and configuration
  6. Pilih platform Web ()
  7. Copy konfigurasi dan masukkan di bawah ini

⚠️ PENTING:

Anda perlu memasukkan API Key dan Project ID yang valid dari Firebase Console. Konfigurasi default di atas adalah placeholder dan tidak akan berfungsi.

ℹ️ Informasi Database Anda:

Database URL: https://pengumuman-nilai-default-rtdb.asia-southeast1.firebasedatabase.app

Project ID yang terdeteksi: pengumuman-nilai

✅ API Key Sudah Disediakan:

API Key telah diisi secara otomatis. Anda hanya perlu memastikan Project ID dan App ID sesuai dengan project Firebase Anda.
